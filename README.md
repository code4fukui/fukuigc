# Fukui Gourmet Collections

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A web app showcasing recommended local dishes and menus in Fukui, Japan.

## Demo
https://code4fukui.github.io/fukuigc/

## Features
- Displays a randomly shuffled gallery of signature dishes from 56 local restaurants.
- Clicking an image navigates to a static map showing the locations of the featured establishments.

## Data Sources
The application uses open data from:
- [Fukui Prefecture Open Data Portal](http://www.pref.fukui.lg.jp/doc/toukei-jouhou/opendata/)
- [ISTS Fukui Local Organizing Committee](https://ists-fukui.pref.fukui.lg.jp/)
