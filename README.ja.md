# Fukui Gourmet Collections

福井県のおすすめのご当地グルメやメニューを紹介するWebアプリです。

## デモ
https://code4fukui.github.io/fukuigc/

## 機能
- 県内56店舗の飲食店の看板メニューをランダムにシャッフルしてギャラリー表示します。
- 画像をクリックすると、掲載店舗の場所を示す静的マップに遷移します。

## データソース
本アプリは以下のオープンデータを使用しています：
- [福井県オープンデータポータル](http://www.pref.fukui.lg.jp/doc/toukei-jouhou/opendata/)
- [ISTS福井実行委員会](https://ists-fukui.pref.fukui.lg.jp/)
